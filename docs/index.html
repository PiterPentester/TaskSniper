About project

TaskSniper - Telegram-bot, written on Python3 with a database on sqlite3. The bot divides tasks by priority into red, yellow and green. To create red priority tasks, use the "@" marker. For yellow - "*". For green - "$". For example, the red priority task "clean in the room" is created like this - "@clean in the room." The reply keyboard is used to interact with the bot, which simplifies the use of the bot. After the "/start" command, the bot will ask what you want to do, create a task, or view? In create mode you can create tasks. In view mode you can view and delete your tasks.

Features

Keep focus

Do not hold the tasks in your head - delegate it to the bot

Split and conquer

Split your tasks by priority and the main tasks will always be met

Use reply keyboard

Do not waste time typing, just use the buttons
